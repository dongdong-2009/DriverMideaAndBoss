所有数据按Byte来返回

Bool:占一个字节，0：False,1：True
Byte:占一个字节
UShort:占二个字节
Int32:占四个字节
Float:占四个字节
Double:占八个字节
String:占2+N个字节，前面2个字节用于表示长度,String用UTF8编码,即最长支持65535长度读取

读取指定类型数据格式。

注:除字符串外,其他类型只须要读取一次.
   读取字符串数据,须先送一帧获取字符串长度的命令,再读取数据(目的:防止字符串过长,不进行长度判断的话,可能导致读取不完全)

格式一:读取指定类型(除字符串以外的类型),指定长度的数据
	"Read"+分段符+
	"Time"+分隔符+"实际时间[2015-12-30 12:12:12]"+分段符+
	"Type"+分隔符+"数据类型"+分段符+
	"Start"+分隔符+"数据起始点"+分段符+
	"End"+分隔符+"数据结束点"
举例:
	11:00:38.5851 [63]  :Read	Time 2016-01-04 11:00:38	Type System.Boolean	Start 0	End 1
	11:00:57.9608 [63]  :Read	Time 2016-01-04 11:00:57	Type System.Int32	Start 0	End 1
	11:01:20.5064 [63]  :Read	Time 2016-01-04 11:01:20	Type System.Double 	Start 0	End 1
	11:02:24.4301 [63]  :Read	Time 2016-01-04 11:02:24	Type System.UInt16 	Start 0	End 1
	11:02:41.2884 [63]  :Read	Time 2016-01-04 11:02:41	Type System.Single 	Start 0	End 1
	11:03:03.5542 [63]  :Read	Time 2016-01-04 11:03:03	Len  System.String 	Start 0	End 1
回送格式:
	"Send"+分段符+
	"Time"+分隔符+"接收时间[即接收数据字符串中间的解析时间]"+分段符+
	"Data"+分隔符+"实际数据[按数据类型所占用Byte来切割数据]



格式二:读取指定长度的字符串将返回的长度
	"Read"+分段符+
	"Time"+分隔符+"实际时间[2015-12-30 12:12:12]"+分段符+
	"Len"+分隔符+"数据类型"+分段符+
	"Start"+分隔符+"数据起始点"+分段符+
	"End"+分隔符+"数据结束点"
举例:
	11:03:03.5542 [63]  :Read	Time 2016-01-04 11:03:03	Len  System.String 	Start 0	End 1
回送格式:
	"Send"+分段符+
	"Time"+分隔符+"接收时间[即接收数据字符串中间的解析时间]"+分段符+
	"Len "+分隔符+"应当接收字符串的长度"



格式三:读取指定长度的字符串
	"Read"+分段符+
	"Time"+分隔符+"实际时间[2015-12-30 12:12:12]"+分段符+
	"Type"+分隔符+"数据类型"+分段符+
	"Start"+分隔符+"数据起始点"+分段符+
	"End"+分隔符+"数据结束点"
回送格式:
	"Send"+分段符+
	"Time"+分隔符+"接收时间[即接收数据字符串中间的解析时间]"+分段符+
	"Data"+分隔符+"实际数据[2字节长度+UTF8编码的字符串]
